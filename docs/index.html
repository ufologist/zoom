    zoom.js

    等比例缩放适配

    功能

    • 让元素等比例缩放以撑满其父容器的宽或者高
    • 让元素缩放之后是水平垂直居中于父容器的

    使用方法

    推荐将 dist/zoom.js 内嵌到 <head> 中

    示例

    // 要适配的元素
var element = document.querySelect('.foo');
// 等比例缩放适配
new Zoom(element);
